# Design

This is an evidence packet, not a feature packet.

The run uses the existing maintained root matrix so the result is comparable to
the previous dry-run and preregistration checks. Report interpretation follows
the literature-backed analyzer discipline:

- treat findings as review triage;
- compare root classes and grouped regions across several owners;
- prefer repeatable false positives/false negatives over one-off intuition;
- keep raw evidence paths local and reproducible.

If the evidence shows a concrete analyzer defect, open or apply a separate
narrow implementation packet.
