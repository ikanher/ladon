# Ladon Proof Region Atlas Overlay

## Why

Flat findings are useful, but the BIFR run shows related signals: broad import
closure, repeated declaration families, proof-family similarity candidates, and
packet evidence. Reviewers need these grouped into regions.

## What

- Add deterministic review regions to the report payload.
- Group import pressure, proof-family repetition/similarity, and packet evidence.
- Render regions in text reports.

## Non-Goals

- No visual graph renderer yet.
- No semantic proof-dependency claim.
- No replacement for raw findings.
