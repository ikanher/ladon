# Proposal

Extend Ladon's local literature corpus with sources for the next atlas and
proof-architecture intelligence packets.

# Scope

- Search online for graph/query, architecture-evolution, clone/smell
  visualization, and Lean/proof-engineering sources.
- Prefer arXiv e-print tarballs when available.
- Update `paper/ladon-code-quality/manifest.json`.
- Update `paper/ladon-code-quality/INDEX.md` with reading-order changes and
  design implications.

# Non-Goals

- No analyzer feature changes.
- No claims that every paper has been deeply read.
- No non-local bibliography service.
